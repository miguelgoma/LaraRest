# Laravel - Docker - Rest

Example of Api Rest with docker services in Laravel 7x

## Installation

1.- Clone this repo
2.- docker-compose up
3.- Go into folder root and execute
    php artisan migrate
    php artisan db:seed

## Usage

POSTMAN

https://www.getpostman.com/collections/bd41a710b7541d9610e2

## Docker containers

docker exec -ti admin_api sh
docker exec -ti admin_db sh
